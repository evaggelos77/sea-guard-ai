# EV SEA GUARD AI

Ψηφιακή υποδομή θαλάσσιας προστασίας για τον λαγοκέφαλο (Lagocephalus sceleratus)
και τα χωροκατακτητικά ξενικά είδη στις ελληνικές θάλασσες.

> **V2 — Πραγματικά δεδομένα.** Τα Risk Score δεν είναι πλέον demo νούμερα.
> Υπολογίζονται ζωντανά από πραγματική θερμοκρασία θάλασσας και πραγματικές
> καταγραφές του είδους.

## Πηγές δεδομένων (ζωντανές, χωρίς κλειδί, απευθείας από browser)

- **Open-Meteo Marine API** — θερμοκρασία επιφάνειας θάλασσας (SST), ρεύματα,
  ύψος κύματος + πρόγνωση 72h ανά ζώνη. <https://open-meteo.com>
- **GBIF Occurrence API** — πραγματικές καταγραφές του είδους
  *Lagocephalus sceleratus* (iNaturalist research-grade, μουσεία, ΕΛΚΕΘΕ κ.ά.).
  taxonKey **2407758** · <https://www.gbif.org/species/2407758>

Και οι δύο πηγές υποστηρίζουν CORS και δουλεύουν χωρίς API key, οπότε η εφαρμογή
παραμένει καθαρά frontend (φθηνό/δωρεάν στατικό deploy).

## Ο μηχανισμός κινδύνου (διαφανής, χωρίς «μαύρο κουτί»)

```
Risk = 45% · Καταλληλότητα SST  +  35% · Πραγματική παρουσία (GBIF)  +  20% · Σήμα κοινότητας
```

- **Καταλληλότητα SST** — ο λαγοκέφαλος είναι θερμόφιλο λεσσεψιανό είδος.
  Καμπύλη καταλληλότητας με κορύφωση ~22–28°C, περιορισμό κάτω από ~14°C.
- **Πραγματική παρουσία** — λογαριθμικά κανονικοποιημένος αριθμός καταγραφών GBIF
  σε τοπικό «κουτί» γύρω από κάθε ζώνη (έμφαση στην τελευταία 3ετία).
- **Σήμα κοινότητας** — αναφορές πολιτών + κιλά αλιέων + δαγκώματα στην περιοχή
  (ανανεώνεται ζωντανά καθώς μπαίνουν νέες αναφορές).

Η **πρόγνωση 24/48/72h** οδηγείται από την πραγματική πρόγνωση SST του Open-Meteo,
άρα μεταβάλλεται με τα πραγματικά δεδομένα — δεν είναι σταθερό μοτίβο.

Όλος ο κώδικας του μηχανισμού: `src/data/seaData.js` (πλήρως σχολιασμένος).

## Τι περιλαμβάνει

- **14 ζώνες** εθνικής κάλυψης (Ρόδος, Κως, Κρήτη, Κάρπαθος, Κάλυμνος, Σάμος, Χίος,
  Λέσβος, Κυκλάδες, Μεσσηνία, Λακωνία, Χαλκιδική, Αττική, Κέρκυρα) με ζωντανό Risk Score.
- **Χάρτης Ελλάδας** με τις ζώνες + **πραγματικές καταγραφές GBIF** ως σημεία.
- **Μπάρα ζωντανών δεδομένων** (κατάσταση, SST επιλεγμένης ζώνης, αριθμός καταγραφών,
  χρόνος, κουμπί ανανέωσης).
- Ρόλοι: **πολίτης / λουόμενος, ψαράς, δήμος/περιφέρεια, admin**.
- **Αναφορά λαγοκέφαλου** με φωτογραφία, GPS, κατάσταση, δάγκωμα, σχόλιο.
- **Panel ψαρά** για κιλά, GPS, ζημιές.
- **Dashboard φορέα** με hotspots, εξαγωγή CSV/PDF.
- **Admin panel** έγκρισης/απόρριψης αναφορών.
- **Αρχική με ασφάλεια**: προειδοποίηση τοξίνης, πρώτες βοήθειες, τηλέφωνα έκτακτης ανάγκης.
- PWA manifest (mobile-first).

## Βασική αρχή / αξιοπιστία

Η πλατφόρμα **δεν** ισχυρίζεται ότι ο δορυφόρος βλέπει έναν-έναν τους λαγοκέφαλους.
Συνδυάζει πραγματικά θαλάσσια δεδομένα, πραγματικές επιστημονικές καταγραφές και
επιβεβαιωμένες αναφορές για να εκτιμά **περιοχές αυξημένης πιθανότητας παρουσίας**.
Δεν αντικαθιστά την επίσημη ενημέρωση — σε επείγον καλέστε **112 / ΕΚΑΒ 166 / Λιμενικό 108**.

## Επόμενη φάση ενσωμάτωσης

- Copernicus Marine (χλωροφύλλη, βιογεωχημικά)
- EMODnet Bathymetry (βάθος, κλίση βυθού, απόσταση ακτής)
- ELNAIS / ΕΛΚΕΘΕ + iSea (επιστημονική βάση ξενικών ειδών, citizen science)
- Vision AI αναγνώριση φωτογραφίας (αντί demo πιθανότητας)
- Backend PostgreSQL + PostGIS για παραγωγικές αναφορές

```powershell
npm install
npm run dev      # http://127.0.0.1:8220
npm run build    # παραγωγικό bundle στο dist/
```
